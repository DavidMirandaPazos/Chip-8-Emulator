Both part 1 and 2 of the assignment are completed.

To play FlighRunner which is the precompiled game simply click on BIN/CS315Emulator.exe. The controls are as follow:
A: Go Down
W: Go Up
S: Go Right
R: Go Left

In order to play any other game included, open the solution and at the file CS315Emulator.cpp line 9 change the name of the program, compile and run. IMPORTANT: The program .ch8 must be inside the data folder.

TETRIS:
W: Move left.
E: Move right.
Q: Rotate.

WIPEOFF: IMPORTANT: This plays sounds.
Q: Move left.
E: Move right.

invaders: IMPORTANT: This plays sounds.
Q: Move left.
W: Shoot.
E: Move right.
